# Kraube API — Brandbook

## Mood

Light, friendly, approachable — but with sharp edges. The brand conveys technical confidence without heaviness. Geometric precision meets warmth.

## Font

**Alegreya Sans SC** — primary brand font. Used for headings, logo text, and hero sections. Small caps give a distinctive, confident character.

- Headings: Alegreya Sans SC, 700-800 weight
- Body text: Inter, 400-600 weight
- Code: SF Mono, Fira Code, Cascadia Code (system monospace)

## Logo

The mascot is a pixel-art robot in blue-gray tones — geometric, friendly, with straight angles.

### Usage Rules

- Product name in prose: **Kraube API** (capital K, capital A)
- Package/code: `kraube` (lowercase, no "API")
- Module path: `github.com/scott-walker/kraube-api`
- CLI binary: `kraube`
- Never use: "kraube-go", "KraubeAPI", "kraube api"
